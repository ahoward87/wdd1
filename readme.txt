# Deployment Plan

## Staging Server

1. Create first (staging) server
2. Install and configure git and apache2 on staging server
3. Create two branches for staging server
   1. Staging branch pushes to staging server for any pre-production testing
   2. A Master branch for all live pushes to server
4. Create local files for GitHub syncing
5. Make sure/test that GitHub is syncing correctly
6. Test all commits before pushing to the Master branch of the server
